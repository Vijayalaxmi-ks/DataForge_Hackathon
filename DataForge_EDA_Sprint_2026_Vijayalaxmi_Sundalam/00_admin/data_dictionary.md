# Superstore Sales Data Dictionary

| Column Name | Data Type | Meaning / Description |
| :--- | :--- | :--- |
| Row ID | Numerical (Integer) | Unique identifier for each row |
| Order ID | Categorical (String) | Unique identifier for each transaction/order |
| Order Date | Date | The date the order was placed |
| Ship Date | Date | The date the order was shipped |
| Ship Mode | Categorical | Shipping method used (e.g., Standard Class, Second Class) |
| Customer ID | Categorical (String) | Unique identifier for the customer |
| Customer Name | Categorical (String) | Name of the customer |
| Segment | Categorical | Customer market segment (e.g., Consumer, Corporate, Home Office) |
| Country | Categorical | Country where the sale occurred |
| City | Categorical | City where the sale occurred |
| State | Categorical | State where the sale occurred |
| Region | Categorical | Geographic region (e.g., East, West, Central, South) |
| Product ID | Categorical (String) | Unique identifier for the product |
| Category | Categorical | Main product category (e.g., Furniture, Office Supplies) |
| Sub-Category | Categorical | Detailed product sub-category (e.g., Chairs, Paper, Phones) |
| Product Name | Categorical (String) | Title/Name of the product |
| Sales | Numerical (Float) | Revenue generated from the sale |
| Quantity | Numerical (Integer) | Number of items purchased |
| Discount | Numerical (Float) | Discount percentage applied to the product |
| Profit | Numerical (Float) | Net profit or loss generated from the sale |
