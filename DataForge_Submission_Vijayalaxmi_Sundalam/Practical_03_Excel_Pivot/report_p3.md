# Practical 3 - Interactive Pivot Dashboards
**Author:** Vijayalaxmi_Sundalam
**Date:** 2026-06-30

## Problem Statement
Stakeholders require a dynamic dashboard layout to interpret multi-dimensional transaction summaries efficiently without scrolling through raw sheets.

## Procedure
1. Isolated data in `dataset_cleaned` and calculated descriptive summary statistics.
2. Built three independent Pivot Tables mapping metrics by Region, Category, and Month.
3. Linked Column, Pie, and Line charts directly to the active pivot tables.
4. Integrated interactive slicer toggles mapped to the entire dataset range to establish dynamic dashboard connections.

## Outputs Created
* `p3_excel_eda_dashboard.xlsx` (with summary stats, pivots, and charts)
* `dashboard_preview.png` (full system dashboard interface snapshot)
